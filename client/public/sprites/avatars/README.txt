将领头像图片目录（PNG）
============================

放置规则
--------
1. 主规则：文件名 = 图鉴 id + ".png"，放在本目录。
   例如：liu_bei.png、zhang_jiao.png、npc_001_程昱.png
2. 备选：若无 catalogId（如部分杂兵），可尝试 name_姓名.png，
   例如：name_黄巾刀兵.png（与场上显示姓名一致）

游戏会按顺序请求上述路径；若均不存在，则显示字头像占位。

如何批量得到文件名列表
----------------------
在项目 client 目录执行：
  npm run avatars:list-ids
将输出所有图鉴 id，便于你对照准备图片。

建议
----
• 个人练习：可自制简笔头像、AI 生成（注意平台条款）、或购买可商用素材包。
• 统一尺寸建议：正方形 96×96～192×192，透明底更佳；游戏内会 crop 为圆角方图。
